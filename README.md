# Frontend Practice — HTML / CSS / JS

Набор проектов, которые я делал для закрепления навыков вёрстки и анимации. Каждый проект — отдельная маленькая задача: попробовать технику, разобраться как оно работает изнутри, и довести до состояния когда смотришь и думаешь «окей, круто».

Никаких фреймворков — только HTML, CSS и немного JavaScript там где без него никак.

---

## Проекты

### 1. 3D переворачивание карточки

Хотел разобраться как работает настоящий 3D-флип без JavaScript. Оказалось всё решается через `transform-style: preserve-3d` и скрытый чекбокс — кликаешь на переключатель, CSS сам крутит карточку на 180°. Добавил переключатель языка EN/RU в том же стиле.

**Что изучил:** `perspective`, `backface-visibility`, CSS-трюк с checkbox + label вместо JS, анимация `border-radius` для blob-эффекта.

---

### 2. Caveman 404 Page

Страница ошибки 404 с двумя первобытными людьми, которые колотят друг друга дубинами. Весь проект — чистый CSS, ни строчки JavaScript. Персонажи собраны из `div`-ов с `border-radius`.

**Что изучил:** как лепить сложные фигуры только из CSS, синхронизация нескольких `@keyframes`, `animation-delay` для поочерёдных движений.

---

### 3. Анимированная форма входа

Форма входа с персонажем, который закрывает глаза когда печатаешь пароль. Персонаж нарисован на чистом CSS — голова, волосы, уши, руки. При фокусе на поле пароля руки поднимаются и закрывают глаза, рот меняется на смущённый. Есть переключатель EN/RU.

**Что изучил:** анимация CSS-персонажей через `transform` + `transition`, `cubic-bezier` для пружинистого движения, связка событий `focus`/`blur` с классами.

---

### 4. Слайдер путешествий

Слайдер с 6 локациями — карточки раскладываются как веер, активная растягивается на весь экран. Автоматическая смена через таймер с индикатором прогресса. Добавил переключатель EN/RU — все тексты, навбар и подписи карточек переключаются.

**Что изучил:** GSAP 3 (анимации, keyframes, Promise-цепочки), предзагрузка изображений, динамическая генерация DOM из массива данных.

---

### 5. Жуткая форма входа

Форма входа где персонаж реагирует на попытки авторизации видео-роликами. Введёшь правильный пароль — одна реакция, ошибёшься — другая, ошибёшься снова — третья. Пароль: `user1` / `1111`.

**Что изучил:** управление `<video>` через JS, счётчик состояний, `setTimeout` для имитации задержки сервера.

---

### 6. Индикатор навигации

Навбар где активный пункт отмечается плавающим индикатором с анимацией. При переключении между пунктами индикатор не просто перепрыгивает — он растягивается, проезжает как луч и схлопывается на новом месте. Эффект сделан через SVG и CSS-переменные.

**Что изучил:** GSAP keyframes с CSS-переменными, динамическое создание SVG в JS, `getBoundingClientRect()` для точного позиционирования.

---

### 7. Кнопка выхода

Кнопка «Log Out» с анимированным персонажем внутри — человечек выбегает через дверь. Две версии: на светлом и тёмном фоне. Всё построено на SVG-путях.

**Что изучил:** сложные SVG-анимации, несколько слоёв SVG в одном элементе, CSS hover-состояния для многосоставной графики.

---

### 8. Карточка продукта

Карточки кроссовок с hover-анимацией — при наведении кроссовок вылетает вверх, снизу появляется блок с выбором размера и цвета. Две карточки, всё на CSS без JavaScript.

**Что изучил:** `overflow: hidden` + `transform` для reveal-эффектов, организация слоёв внутри карточки.

---

### 9. Кнопка «Добавить в корзину»

Кнопка с последовательной анимацией: футболка выезжает снизу → деформируется → падает в корзину → корзина уезжает → возвращается текст. Две темы — светлая и тёмная.

**Что изучил:** GSAP MorphSVGPlugin (деформация SVG-путей), многоэтапные keyframe-анимации, блокировка повторного клика через флаг состояния.

---

### 10. Кнопка заказа футболки

Нажимаешь кнопку — футболка складывается, загружается в пушку и вылетает со звуком. Текст меняется побуквенно. При следующем нажатии всё сбрасывается.

**Что изучил:** GSAP Timeline с вложенными Timeline-ами, Splitting.js для посимвольной анимации, `Audio API`, CSS HSL-переменная для динамической смены цвета.

---

### 11. Переключатель день / ночь

Переключатель тёмной темы в виде дроида BB-8 из Звёздных войн. Весь персонаж — чистый CSS: сфера, антенны, полосы, тень. JavaScript — одна строка: `classList.toggle('dark')`.

**Что изучил:** как создать сложного персонажа только из CSS, `mix-blend-mode`, переключение темы через класс на `body`.

---

### 12. Форма входа со сменой фона

Форма входа на фоне сменяющихся фотографий. Десять изображений перелистываются автоматически через CSS-анимацию, без JS.

**Что изучил:** `@keyframes` с `background-image`, синхронизация анимации фона с контентом поверх него.

---

### 13. Форма входа с люстрой

Наверное самый атмосферный проект в этой подборке. Форма входа в тёмной комнате — люстра нарисована на SVG (~230 строк), качается как настоящий маятник, кристаллы мерцают. Ползунок яркости буквально «включает свет»: чем больше крутишь, тем ярче горят лампочки, больше искр и лучше видна форма.

**Что изучил:** SVG-фильтры (`feGaussianBlur`, `feFlood`), физика маятника в `requestAnimationFrame`, управление CSS-переменными из JS в реальном времени, генерация и удаление DOM-элементов (искры).

---

## Стек

Всё на ванильном HTML/CSS/JS. Где нужна была анимационная мощь — использовал **GSAP 3**. Больше никаких зависимостей.
